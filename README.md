# 🚀 QuickRead

QuickRead is an AI-powered news platform that delivers the latest news with intelligent summaries, blogs on recent developments, and real-time AQI (Air Quality Index) updates.

Built using Next.js and modern web technologies, QuickRead helps users stay informed in a fast, clean, and minimal way.

---

## 🌟 Features

📰 Latest News Headlines  
🤖 AI-Generated News Summaries  
✍️ Blogs on Recent Developments  
🌍 Real-Time AQI Checker  
⚡ Fast and Responsive UI  
🔐 Secure API Handling using Environment Variables  

---

## 🛠️ Tech Stack

**Frontend:**
- Next.js
- React.js
- Tailwind CSS

**Backend / APIs:**
- News API
- AI API (for summarization)
- AQI API

**Deployment:**
- Vercel
